About

The application used to build this website is Spring Boot, and it is fully coded and built with Java, HTML, and Maven. Although I used some code templates from peers to help me get started with coding the web application, the majority of this project is coded by me. I used Amazon Web Services to deploy this project to the internet, using AWS EC2 and Elastic Beanstalk. This website is a work in progress, meaning that many features/projects are not finished.
