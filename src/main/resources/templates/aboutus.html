About

Hello viewer! My name is Shekar Krishnamoorthy and this is my personal website. I built this website to express my creative side of designing a website, while also showcasing my coding skills. To create the tomcat server for the web application, I used Maven and Java Springboot, and to build the individual components of the application, I used Java, HTML, CSS, and a bit of SQL. Although I used code templates from peers and teachers to help me get started with building the website, the majority of this project is coded by me. And finally, in order to make this website available to the public, I used an EC2 and Elastic Beanstalk from Amazon Web Services. This project is still a work in progress, meaning that some features may be broken, and many more projects are on the way soon! Feel free to explore the website!
